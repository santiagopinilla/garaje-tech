# 🌙 Garaje Tech — Gadgets & Periféricos Premium

Landing page de una tienda de tecnología con diseño **premium en tema oscuro** (con toggle a tema claro): teclados Epomaker, almacenamiento Orico, carga rápida Ugreen y periféricos de las mejores marcas.

## ✨ Características

- Tema oscuro elegante por defecto, con **botón para alternar a tema claro** (la preferencia se guarda en `localStorage`)
- Identidad visual con degradado violeta → rosa en botones, textos y promociones
- Hero con collage de fotos flotantes animadas y chips de promoción
- 12 productos con fotos reales (Unsplash) y marcas de ejemplo: Epomaker, Orico, Ugreen, Logitech, Redragon, HyperX, Xbox y Baseus
- Buscador en vivo + filtros por categoría combinados
- Carrito funcional con contador animado y notificaciones toast
- Cupón de descuento con **click para copiar** al portapapeles
- Sección de opiniones de clientes, beneficios y newsletter
- Animaciones al hacer scroll y diseño responsive

## 🚀 Uso

No requiere dependencias ni build. Solo abre `index.html` en el navegador, o sirve la carpeta con cualquier servidor estático:

```bash
npx serve .
```

> Nota: las fotos de productos se cargan desde el CDN de Unsplash, por lo que se necesita conexión a internet (incluye un placeholder de respaldo si alguna foto falla).

## 🛠️ Tecnologías

HTML5 · CSS3 (variables para theming, grid, animaciones) · JavaScript vanilla
